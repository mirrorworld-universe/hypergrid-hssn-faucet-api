# HyperGrid Faucet API

This Faucet API endpoint allows HyperGrid Testent Grid operatrs to request SOL for their users / wallets.

## Install pm2

```
npm i -g pm2
```

## Install Dependencies

```
npm i
```

## Dev

```
npm run dev
```

## Build

```
npm run build
```

## Run Scripts on Server

```
pm2 start
```

## Check Scripts Logs on Server

```
pm2 logs
```
