# HyperGrid HSSN Faucet API

This Faucet API endpoint allows testnet HSSN providers to claim SOL for the HSSN Nodes.

## Install pm2

```
npm i -g pm2
```

## Install Dependencies

```
npm i
```

## Dev

```
npm run dev
```

## Build

```
npm run build
```

## Run Scripts on Server

```
pm2 start
```

## Check Scripts Logs on Server

```
pm2 logs
```
